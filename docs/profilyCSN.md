# Dokumentace k funkci profilyCSN

## Popis
Funkce `profilyCSN` slouží k zpracování různých typů profilů v rámci aplikace.

## Parametry
Funkce podporuje následující typy profilů:

- **PLO**: Popis pro PLO
- **OBD**: Popis pro OBD
- **KR**: Popis pro KR
- **TRKR**: Popis pro TRKR
- **4HR**: Popis pro 4HR
- **6HR**: Popis pro 6HR
- **I**: Popis pro I profily
- **TR4HR**: Popis pro TR4HR

## Příklady použití
```python
# Příklad 1
výsledek = profilyCSN('PLO')
print(výsledek)

# Příklad 2
výsledek = profilyCSN('OBD')
print(výsledek)
```
